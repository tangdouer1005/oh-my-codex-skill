# Output Format

Use this structure both in the assistant response and in the saved Markdown artifact.

At the top of the saved Markdown, add:

- `Source summary file`: absolute path
- `Source query file`: absolute path
- `Generated by`: `report-benchmark-groundtruth-builder`
- `Output time`: timestamp if available

## 1. Run Scope

Write 3 to 5 bullets:

- which query ids were processed
- which queries were skipped and why
- what output directory was used
- whether all artifacts were generated successfully

## 2. Per-Query Groundtruth

For each processed query, use this structure:

### Query N
- `Query`: the exact selected query text
- `Chart type`: expected chart form
- `Direct source data paths`: paths to the saved direct-data artifacts
- `Final chart data paths`: paths to the saved final-data artifacts
- `Chart code path`: path to the Python script
- `Chart image path`: path to the rendered image
- `Direct source data summary`: a short summary of what direct values were extracted
- `Reasoning steps`: a numbered list of the transformation steps used to reach the final data
- `Final chart-ready data summary`: a short summary of the final plotted records
- `Validation`: one short line confirming whether code execution and image generation succeeded
- `Open issues`: one short line, or `none`

When summarizing the JSON artifacts, explicitly state whether quantitative fields are stored as numbers with separate unit fields.

## 3. Completion Check

List concise bullets covering:

- direct data saved
- final data saved
- code saved
- image saved
- Markdown saved
